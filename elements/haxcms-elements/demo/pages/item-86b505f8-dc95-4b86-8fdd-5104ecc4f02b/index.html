Open community, Penn State and Beyond. Free and Open means that we are an Open Community, welcoming to all, Penn State and beyond. It means that we value the 5Rs of content reuse in an OER context and extend it to the open web more broadly for non-academic websites as well.

Ways we are committed to Free and Open

  • Open Documentation
  • Open APIs and public infrastructure in HAX and the web in general
  • Self-hosting, static hosting, SaaS, PaaS, and vendor community will all be on an equal playing field. User content is to remain portable,
  • Offline capable, remixable, downloadable, clone-able, and generally ubiquitous to the greatest extent possible
  • No lock-in to platform or editor. Maximal support for one-click migrations via public URL references
  • Block made to work with HAX are opt-in and should work with or without the editor present (Example)
  • No censorship on usage
  • Apache 2.0 license on HAX core and all HAX integrations where possible (open licenses to mirror community integrations as needed. Example: Drupal is GPL)
  • App supplier specification is open and code to generate "App Store" definitions is publicly available
