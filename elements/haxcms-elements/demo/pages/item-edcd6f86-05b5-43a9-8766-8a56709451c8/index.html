HAX seeks to leverage web technologies in an ecologically responsible manner. Our community values..

  • Turning to web standards over large libraries to use less resources
  • Enabling offline capable progressive web app capabilities to reduce energy usage
  • Tuning caching and usage settings to leverage less energy through redundant traffic requests
  • Taking a database-less approach to reduce complexity, portability, and require less "always on" approaches to technology
  • Leveraging micro-services whenever possible in configurations that only run when necessary for processing API requests

If you have areas for possible improvement in energy usage by the HAX platform please submit an issue request to our queue detailing how we can achieve a greener platform.
