This is a list of the data formats that HAX supports for entry and multi-media presentation

  • MathJAX
  • Latex
  • Markdown / md (convert to HTML or embed source as HTML)
  • DocX (convert to HTML)
  • Web Components / HTML (duh)
  • Images (png, jpg, jpeg)
  • Video (mp4)
  • Audio (mp3 file or via browser based recording)
