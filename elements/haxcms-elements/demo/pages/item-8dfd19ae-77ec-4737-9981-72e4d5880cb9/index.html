Getting HAX setup in BackdropCMS

  1. Search for HAX on BackdropCMS.org or from within your Backdrop site
  2. Follow the instructions in the Web components module README for building
    • Don't worry, HAX requires Web components module but the module comes with files that supply details to install HAX
  3. Enable the HAX and Web components modules
  4. Go to a node in your site, you should see a tab that says "HAX mode"
  5. Enjoy HAXing your BackDropCMS site
