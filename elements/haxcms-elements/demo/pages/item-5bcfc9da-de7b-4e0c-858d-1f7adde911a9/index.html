These tools can be used to convert documents between different formats

  • DocX to HTML
  • DocX to PDF
  • Markdown to HTML (and HTML to Markdown)
  • Compression / image resizer (tweak settings then right click to download)
  • Screenshot any URL
  • Secure feedback

These tools are provided as part of our microservice infrastructure. Have ideas / needs for other types of simple converters? Join our Discord and ask!
