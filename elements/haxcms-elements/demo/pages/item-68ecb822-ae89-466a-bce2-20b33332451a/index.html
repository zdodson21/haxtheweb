Meet ups

We have a monthly meet-up for core developers / contributing groups as well as a bi-monthly meet-up for our end users. Details on how to join either can be found by hopping onto our discord channel to chat.

Want to host a HAX camp?

HAX Camp is a chance to learn and develop web components for the HAX ecosystem and beyond! Because HAX is built on the W3C specification Web Components, by improving the broader web you are learning to write components for use with the HAX authoring system!
